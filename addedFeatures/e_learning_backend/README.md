

## Description
This project is a starter kit for our ( Masbah & Shoaib ) future freelancing agency. Masbah is responsible for this backend system and Shoaib is responsible for the frontend system.
This project is designed with Nest Framework.
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod

# initialize database
$ yarn prisma_build

# generate prisma database studio
$ yarn prisma studio
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
